# ZenDo: Task Management Mobile Application
Tools: Flutter, Dart, SQLite, REST API
- Engineered a fully functional cross-platform Android task management application using Flutter and
Dart, implementing complete CRUD operations with a normalized two-table SQLite relational database
schema
- Implemented SHA-256 password hashing for user authentication, ensuring passwords are never stored
in plaintext and demonstrating security-conscious development practices
- Integrated a three-tier REST API fallback chain connecting to three external quote APIs sequentially,
with graceful degradation to hardcoded local content ensuring zero downtime for users
- Designed a layered application architecture separating presentation, business logic, data access, and
external service layers
